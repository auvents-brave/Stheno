# Stheno

> **Stheno** (Greek: Sthenno), one of the three Gorgon sisters in Greek mythology, traditionally represents strength and endurance.

Stheno is a small cross-platform Swift package that consolidates reusable, context-independent components with clear APIs and strong test coverage.

Continuous Integration (CI) is handled with GitHub Actions, including CodeQL analysis.
